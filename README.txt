Para el uso de la herramineta es ncesario disponder del lenguaje python en la version 3
el código necesita instalar los modulos que se esecifican en "requieriments.txt" y para su
uso en windos simplmente ejecuar el código en el interprete de python, en el caso de sitribuciones 
linux se debe abrir la terminal donde se encuentre el "main.py" y a su vez escribir ./main.py
en el dado caso que no sea posible ejecutarse tal vez el problema se deba a que el archivo no
tiene permiso de ejecuicon por lo que debera dar click derecho, propiedades y activar el permiso 
de ejecuion o tambien uede usar el comando chmod de esta manera "chmod +x main.py" omitiendo las 
comiilas y el archivo tebdrá permiso de ejecucion.
